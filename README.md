# Spotify2MP3

Spotify2MP3 is a simple and easy Python module for downloading songs from Spotify.
Song metadata collected from Spotify is used to search YouTube and download.

See https://developer.spotify.com/ for Spotify API, and information on how to obtain Client ID and Secret.

Libraries used:
* spotipy: https://pypi.org/project/spotipy/
* pytubefix: https://pypi.org/project/pytubefix/
* pydub: https://pypi.org/project/pydub/
* music_tag: https://pypi.org/project/music_tag/
